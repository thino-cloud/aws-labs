# AWS Labs

This repository contains hands-on labs for learning Amazon Web Services.

The goal is to explore core AWS services and understand how cloud infrastructure works.

## Topics Covered

- VPC Networking
- EC2 Compute
- S3 Storage
- IAM Security
- CloudWatch Monitoring

## Lab Categories

### Networking

- VPC Setup
- Subnets
- Security Groups

### Compute

- EC2 Instance Deployment

### Storage

- S3 Buckets

### Security

- IAM Users and Roles

### Monitoring

- CloudWatch Metrics

## Goal

Understand the fundamentals of AWS cloud infrastructure and services.
